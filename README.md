## About

Craftboard is a Vue.js-based dashboard application that integrates with Craft.do to provide a centralized view of your Craft workspace. Create customizable dashboards with widgets for collections, bookmarks, RSS feeds, tasks, daily notes, graphs, and more. Organize your workspace data in a beautiful, responsive interface that works seamlessly on desktop and mobile devices.

**Key Features:**

- 📊 **Customizable Widgets** - Display collections, bookmarks, RSS feeds, tasks, and more
- 📱 **Responsive Design** - Works beautifully on desktop and mobile
- 🎨 **Modern UI** - Clean, intuitive interface with dark/light theme support
- 🔗 **Craft Integration** - Deep integration with Craft.do API
